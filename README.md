# Eagle Buildings — Sticker & Quote Generator (Web App)

A full browser-based version of the desktop app. No server required.

## Files
- `index.html` — Main app shell and tab layout
- `style.css` — All styles
- `images.js` — Logo and default images (base64 embedded)
- `settings.js` — Settings management via localStorage
- `sticker.js` — Window sticker PDF generation
- `batch.js` — CSV import and batch PDF generation
- `quote.js` — Customer quote PDF generation
- `app.js` — App initialization and tab switching
- `vercel.json` — Vercel deployment config

## Deploy to Vercel
1. Push all files to a GitHub repo
2. Connect repo to Vercel
3. Deploy — that's it

## Local Testing
Open `index.html` directly in Chrome or Safari — no build step needed.
